Aplicando estilos em CSS

CSS Inline

estilo CSS INLINE é utilizado quando o design do projeto é feito a estilização na mesma linha de código do documento html, NÃO RECOMENDADO, polui muito o codigo, dificultando a leitura do mesmo, CLEAN CODE.

CSS Interno

Já a estilização de CSS interno, aconteçe quando a estilização é feita atraves da TAG STYLE, aberta no HEAD do documento html, fascilitando assim a estilização do projeto, porem fator negativo, é que os codigos em css geralmente ocupam muito espaço, recomando para projetos menores, de poucas paginas e baixa manutenção personalizada.

CSS Externos

mais utilizado pelos profissionais da área, fascilitando a manutençãoe controle total de toda estilização realizada no projeto. para utilizar esse formato de estilização, basta realizar LINK CSS no head do doc html, abrindo um outro doc com formato css, e linkando ambos, para trabalharem em conjunto no projeto. perfect!
